- `_posts/`: Technical blog articles and experiment logs.
- `assets/`: Static assets and images.
- `projects/`: Project highlights.
- `index.md`: Homepage.
- `blog.md`: Blog index.
- `projects.md`: Project catalog.

## 🛠️ Local Development
To run the blog locally:
```bash
bundle install
bundle exec jekyll serve
```
